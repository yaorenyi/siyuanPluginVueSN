# AI 内容生成器 - 拖拽文档功能

## 功能概述

AI 内容生成器现在支持**拖拽方式加载文档**，可以直接拖拽块、标签页或文档树中的文档到面板中，提供最直观的交互体验。

---

## 支持的拖拽方式

### 1️⃣ 拖拽块

拖拽思源笔记编辑器中的任意内容块到 AI 面板：

```
┌─────────────────┐    拖拽     ┌──────────────────────┐
│  文档内容块      │  ───────→  │  AI 内容生成器面板    │
│  [段落]         │            │                      │
│  [标题]         │            │  ✅ 已加载文档        │
└─────────────────┘            └──────────────────────┘
```

**适用场景：**
- 快速处理当前文档
- 编辑特定段落

### 2️⃣ 拖拽标签页

拖拽思源笔记顶部标签页到 AI 面板：

```
┌─────────────────────────────────────────┐
│ [文档1] [文档2▼] [文档3]  ← 拖拽标签    │
├─────────────────┬───────────────────────┤
│                 │                       │
│  编辑器         │  AI 面板              │
│                 │  ✅ 已加载文档2       │
└─────────────────┴───────────────────────┘
```

**适用场景：**
- 快速切换并处理其他文档
- 多文档并行处理

### 3️⃣ 拖拽文档树

从左侧文档树拖拽文档到 AI 面板：

```
┌──────────┐                ┌──────────────┐
│ 📁 笔记本 │                │  AI 面板     │
│  ├ 文档A  │  ← 拖拽        │  ✅ 已加载   │
│  ├ 文档B  │  ─────────→    │  文档B      │
│  └ 文档C  │                │              │
└──────────┘                └──────────────┘
```

**适用场景：**
- 从文档树快速选择
- 处理特定文件夹中的文档

## 视觉反馈

### 拖拽进入面板

当拖拽内容进入面板区域时：

1. **边框高亮** - 面板显示蓝色虚线边框
2. **背景变化** - 背景颜色根据拖拽类型变化：
   - 拖拽块：浅蓝色 (5%)
   - 拖拽标签页：中蓝色 (8%)
   - 拖拽文档树：深蓝色 (10%)

### 拖拽遮罩层

松开鼠标前显示提示遮罩：

```
┌────────────────────────────────────┐
│                                    │
│        ↕️ (弹跳动画)               │
│      拖拽标签页                   │
│    松开鼠标加载文档               │
│                                    │
└────────────────────────────────────┘
```

**特点：**
- 半透明蓝色背景 (95% 透明度)
- 背景模糊效果 (backdrop-filter)
- 图标弹跳动画
- 清晰的文字提示

## 使用步骤

### 方式一：拖拽块

1. 在思源编辑器中，鼠标悬停在要处理的块上
2. 按住鼠标左键拖动块
3. 拖动到 AI 内容生成器面板
4. 看到蓝色遮罩提示后松开鼠标
5. 文档自动加载到面板

### 方式二：拖拽标签页

1. 找到顶部标签页中要处理的文档
2. 按住标签页拖动
3. 拖动到 AI 面板区域
4. 看到提示后松开鼠标
5. 文档自动加载

### 方式三：拖拽文档树

1. 在左侧文档树中找到目标文档
2. 按住文档项拖动
3. 拖动到 AI 面板
4. 松开鼠标
5. 文档自动加载

## 技术实现

### 事件监听

```typescript
// 监听拖拽事件
@dragenter="handleDragEnter"   // 进入区域
@dragover="handleDragOver"     // 悬停
@dragleave="handleDragLeave"   // 离开区域
@drop="handleDrop"             // 放置
@dragend="handleDragEnd"       // 拖拽结束
```

### 拖拽类型检测

```typescript
// 检测拖拽类型
const isBlock = target.closest('.protyle-wysiwyg__block');  // 块
const isTab = target.closest('.layout__tab');               // 标签页
const isTree = target.closest('.file-tree');                // 文档树
```

### 数据提取

```typescript
// 方式 1: 从 dataTransfer 获取思源数据
const siyuanData = transfer.getData('siyuan/block');

// 方式 2: 从拖拽元素获取信息
transfer.items[0].getAsString((data) => {
  // 提取文档 ID
});

// 方式 3: 从当前激活文档获取
const docId = getActiveDocId();
```

## 与自动加载配合使用

拖拽和自动加载可以配合使用：

| 场景 | 推荐方式 |
|------|----------|
| 处理当前激活文档 | 自动加载 |
| 快速切换到其他文档 | 自动加载 |
| 处理特定块 | 拖拽块 |
| 从文档树选择 | 拖拽文档树 |
| 多文档并行处理 | 拖拽标签页 + 手动模式 |

### 工作流程示例

```
1. 打开 AI 面板（自动加载当前文档）
2. 拖拽其他文档标签页到面板（切换文档）
3. 关闭自动加载开关（专注当前文档）
4. 使用 AI 编辑功能
5. 完成后重新开启自动加载
```

## 常见问题

### Q: 拖拽不生效？
**A:** 确保：
- 拖拽的是思源笔记的块、标签页或文档树
- 拖动到 AI 面板区域内
- 看到蓝色高亮或遮罩提示

### Q: 如何取消拖拽？
**A:** 在遮罩显示时，将鼠标移出面板区域即可取消。

### Q: 拖拽会覆盖已加载的文档吗？
**A:** 会。拖拽会替换当前文档，如需保留请先手动保存或关闭自动加载。

### Q: 拖拽和自动加载哪个优先？
**A:** 拖拽优先。手动拖拽会立即加载文档，不受自动加载影响。
