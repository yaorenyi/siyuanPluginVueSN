# 单词查询功能更新 - 翻译功能与UI优化

## 更新概述

为单词查询面板添加了长文翻译功能，并优化了整体UI界面，提供更好的用户体验。

## 新增功能

### 1. 双模式切换

- **单词查询模式**：原有的单词/词语查询功能
- **长文翻译模式**：新增的文本翻译功能

通过顶部标签页快速切换两种模式。

### 2. 长文翻译功能

#### 功能特点

- 支持多语言翻译（中文、英文、日文、韩文、法文、德文、西班牙文）
- 自动语言检测
- 语言交换功能（一键互换源语言和目标语言）
- 实时翻译状态显示
- 翻译结果复制和导出

#### 使用方式

1. 切换到"长文翻译"标签
2. 在上方文本框输入要翻译的内容
3. 选择源语言和目标语言
4. 点击"翻译"按钮或按 Ctrl/Cmd + Enter
5. 翻译结果显示在下方区域

#### 操作按钮

- **清除**：清空输入和输出内容
- **翻译**：执行翻译操作
- **交换语言**：互换源语言和目标语言（中间圆形按钮）
- **复制**：复制翻译结果到剪贴板
- **导出**：将原文和译文格式化后复制到剪贴板

- 音标/拼音

## 快捷键

- `Ctrl/Cmd + Enter`：执行查询或翻译
- `Escape`：清除结果并关闭下拉菜单
