# 统计功能测试指南

## 测试前准备

1. 确保已构建插件：`npm run build`
2. 重启思源笔记或重新加载插件
3. 打开右侧边栏的"数据统计"面板

## 功能测试清单

### 1. 基础统计卡片 ✅

**测试步骤：**
- 查看顶部的统计卡片
- 验证以下数据是否正确显示：
  - 📓 笔记总数
  - ✍️ 总字数
  - 🧩 内容块
  - 📎 附件
  - 🏷️ 标签
  - 🔗 双链

**预期结果：**
- 所有卡片显示渐变背景色
- 数字格式化正确（千分位分隔）
- 悬停时卡片有上浮效果

### 2. 新增扩展统计 ✅

**测试步骤：**
- 查看扩展统计区域
- 验证以下数据：
  - 📅 今日新增：今天创建的文档数
  - ✏️ 今日修改：今天修改的文档数
  - 📊 平均字数：每文档平均字数

**预期结果：**
- 数据实时更新
- 数字高亮显示（蓝色）

### 3. 视图模式切换 ✅

**测试步骤：**
1. 点击"日"按钮
   - 选择不同天数范围（7天、15天、30天、季度、半年、整年）
   - 验证柱状图和数据列表更新
   
2. 点击"周"按钮
   - 查看最近4周的每周字数统计
   - 验证日期标签格式（MM/DD - MM/DD）
   
3. 点击"月"按钮
   - 选择不同年份范围（最近一年、两年、三年）
   - 查看每月字数统计
   - 验证日期标签格式（YYYY/M）
   
4. 点击"年"按钮
   - 从下拉菜单选择不同年份
   - 查看该年度的月度统计

**预期结果：**
- 切换视图时显示加载状态
- 图表和数据列表同步更新
- 今日数据高亮显示（粉色）

### 4. 柱状图交互 ✅

**测试步骤：**
- 悬停在柱状图的柱子上
- 查看工具提示信息
- 验证今日数据的特殊样式

**预期结果：**
- 悬停时柱子透明度变化
- 显示完整的日期和字数信息
- 今日柱子为粉色渐变，带边框

### 5. 热门标签云 🏷️

**测试步骤：**
- 滚动到"热门标签"区域
- 查看标签显示
- 悬停在标签上
- 点击标签（当前仅有悬停效果）

**预期结果：**
- 标签按使用频率排序
- 标签大小根据使用次数动态调整
- 悬停时标签放大并变色
- 显示使用次数提示

### 6. 最近活跃文档 📝

**测试步骤：**
- 滚动到"最近活跃"区域
- 查看最近修改的文档列表
- 点击文档项

**预期结果：**
- 显示最近5个修改的文档
- 显示文档标题、修改时间、字数
- 时间显示智能化（今天、昨天、N天前）
- 点击后打开对应文档
- 悬停时项目右移动画

### 7. 刷新功能 ✅

**测试步骤：**
- 点击顶部的刷新按钮
- 观察刷新动画
- 验证数据更新

**预期结果：**
- 刷新按钮显示旋转动画
- 数据重新加载
- 最后更新时间更新

### 8. 响应式布局 ✅

**测试步骤：**
- 调整侧边栏宽度
- 验证布局适配

**预期结果：**
- 卡片自动调整大小
- 柱状图横向滚动
- 文字不溢出

## 性能测试

### 1. 加载速度
- 首次打开面板应在 2 秒内完成加载
- 切换视图应在 1 秒内完成

### 2. 数据准确性
- 对比手动统计的数据
- 验证各项统计数字的准确性

### 3. 内存占用
- 长时间使用不应有内存泄漏
- 切换视图多次后性能稳定

## 调试技巧

### 查看控制台日志
```javascript
// 打开开发者工具（F12）
// 查看以下日志：
- "注册统计面板 Dock"
- "统计面板 Dock 初始化完成"
- "获取统计数据失败:" （如果有错误）
```

### 手动触发刷新
```javascript
// 在控制台执行
const panel = document.querySelector('.statistics-panel');
if (panel) {
  panel.__vueParentComponent.exposed.refreshData();
}
```
