# 数据统计功能集成完成

## 功能概述

已成功集成数据统计功能到思源笔记插件，在右侧边栏提供笔记数据的统计和分析。

## 实现的功能

### 1. 基础统计卡片
- 📓 笔记总数
- ✍️ 总字数
- 🧩 内容块数量
- 📎 附件数量
- 🏷️ 标签数量
- 🔗 双链数量

### 2. 多维度时间统计
- **日视图**: 最近7天每日字数（可选择最近一周或一个月）
- **周视图**: 最近4周每周字数
- **月视图**: 指定年份的12个月字数统计
- **年视图**: 最近5年每年字数统计

### 3. 可视化图表
- 渐变色柱状图展示字数变化趋势
- 高亮显示今天的数据
- 支持鼠标悬停查看详细数值
- 响应式设计，自动适配数据量

### 4. 交互功能
- 视图模式切换（日/周/月/年）
- 年份和月份选择器
- 实时数据更新
- 最后更新时间显示

## 使用说明

1. **启用功能**
   - 在插件设置中确保"启用数据统计功能"已开启（默认开启）
   - 重启插件或重新加载思源笔记

2. **打开统计面板**
   - 点击右侧边栏的统计图标（柱状图图标）
   - 或通过超级面板访问统计功能

3. **查看统计数据**
   - 顶部显示6个基础统计卡片
   - 中间提供4种视图模式切换
   - 底部显示详细的时间统计列表

4. **切换视图**
   - 点击"日/周/月/年"按钮切换视图
   - 在月视图和年视图中可选择具体年份
   - 在月视图中可选择具体月份

## 数据说明

- **笔记总数**: 统计所有文档类型的块（type='d'）
- **总字数**: 统计所有段落块（type='p'）的内容长度
- **内容块**: 统计段落、标题、列表、表格等内容块
- **附件**: 统计图片、音频、视频、小工具等
- **标签**: 统计唯一标签数量
- **双链**: 统计引用关系数量

## 注意事项

1. 统计数据基于本地数据库，需要确保数据库完整性
2. 首次打开可能需要几秒钟加载数据
3. 大量数据时建议使用周视图或月视图
4. 字数统计仅包含段落内容，不包含标题、列表等
