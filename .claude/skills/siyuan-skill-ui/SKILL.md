## 代码风格
- 使用 scss 生成样式
- 兼容黑色白色主题
- 响应式设计兼容移动端
- 代码分离：.vue后缀文件有css代码应该在当前功能生成index.scss文件并把代码移动到此文件。通过 <style scoped lang="scss">中引入，使用 @use "./index.scss";

## 审查优化
-  在不影响UI结构的情况下，审查并移除冗余或未使用的 CSS 规则，优先合并重复样式，确保代码简洁、高效且易于维护。
