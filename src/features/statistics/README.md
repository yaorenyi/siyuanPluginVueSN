# 数据统计功能

## 功能说明

数据统计功能提供思源笔记的使用数据统计和分析，帮助用户了解笔记使用情况。

## 主要特性

### 1. 基础统计
- **笔记总数**: 统计所有文档数量
- **总字数**: 统计所有段落内容的字数
- **内容块**: 统计各类内容块数量
- **附件**: 统计图片、音频、视频等附件数量
- **标签**: 统计标签使用数量
- **双链**: 统计引用关系数量

### 2. 多维度时间统计
- **日视图**: 显示最近7天的每日字数统计
- **周视图**: 显示最近4周的每周字数统计
- **月视图**: 显示指定年份的每月字数统计
- **年视图**: 显示最近5年的每年字数统计

### 3. 可视化展示
- 彩色卡片展示基础统计数据
- 柱状图展示时间维度的字数变化
- 高亮显示当天数据
- 支持数据详情列表查看

## 使用方法

1. 在插件设置中启用"数据统计功能"
2. 点击右侧边栏的统计图标打开统计面板
3. 切换不同的视图模式查看统计数据
4. 在月视图和年视图中可选择具体年份和月份

## 技术实现

- 使用思源笔记 SQL API 查询数据库
- 优化查询性能，单次查询获取时间段数据
- 支持中英文国际化
- 响应式设计，适配不同屏幕尺寸

## 数据来源

所有统计数据来自思源笔记本地数据库，包括：
- `blocks` 表：内容块数据
- `refs` 表：引用关系数据
- `spans` 表：标签数据
- `attributes` 表：属性数据
