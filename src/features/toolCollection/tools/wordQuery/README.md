# 单词查询

快速查询单词释义和翻译。注册为右侧边栏 Dock 面板，集成 AI API（从插件设置动态获取配置），支持英文单词查询释义、文本翻译（多语言互译）、自动识别英文单词。

## 已支持功能

| 模式 | 说明 |
|------|------|
| 单词查询 | 查询英文单词/中文词语的详细释义、音标、谐音、例句 |
| 长文翻译 | 自动检测多语言文本互译（中/英/日/韩/法/德/西） |
| 代码翻译 | 中文→英文代码命名翻译，支持驼峰/帕斯卡/下划线等 5 种风格 |
| 代码注释 | 为代码生成 JSDoc/行注释/块注释等风格注释 |
| 代码解释 | 分析代码功能、复杂度及优化建议 |
| 正则生成 | 根据自然语言描述生成正则表达式，支持测试匹配 |

## 配置项

- `enableWordQuery` — 启用/禁用单词查询功能（超级面板开关）

## 扩展建议

以下方向可供未来迭代参考：

1. **发音 API 集成**：接入真实的 TTS API（如 Web Speech API 或第三方服务）替代 `speechSynthesis`，提升发音准确性
2. **历史记录**：持久化查询历史，支持近期查询快速回看
3. **批量查询**：支持粘贴单词列表批量查询并导出为表格
4. **快捷粘贴**：监听剪贴板变化，自动触发查询（适合阅读外文资料场景）
5. **悬浮查询**：集成 `floatingToolbar`，选中文本后一键弹窗查询
