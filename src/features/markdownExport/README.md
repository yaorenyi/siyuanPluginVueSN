# Markdown 导出功能

## 功能概述

一键导出所有笔记本为 Markdown 格式,支持批量导出和选择性导出。

## 功能特点

- ✅ **批量导出**: 一键导出所有笔记本
- ✅ **选择性导出**: 可以选择需要导出的笔记本
- ✅ **进度显示**: 实时显示导出进度
- ✅ **日志记录**: 记录导出成功和失败的信息
- ✅ **ZIP 打包**: 每个笔记本自动打包为 ZIP 文件,包含所有文档和相关资源(图片等)

## 使用方法

### 1. 打开导出面板

点击顶部工具栏的"导出笔记本为 Markdown"按钮(图标: 📥)

### 2. 选择笔记本

在弹出的面板中,勾选需要导出的笔记本。可以使用以下快捷操作:

- **全选**: 选中所有笔记本
- **取消全选**: 取消所有选择

### 3. 开始导出

点击"导出选中的笔记本"按钮,系统会自动:

1. 逐个导出选中的笔记本
2. 显示导出进度
3. 自动下载 ZIP 文件
4. 记录导出日志

### 4. 查看结果

导出完成后:

- 成功导出的笔记本会显示 ✅ 标记
- 失败的笔记本会显示 ❌ 标记,并在日志中显示错误信息

## 导出格式

每个笔记本导出为一个 ZIP 文件,包含:

```
笔记本名称.zip
├── 文档1.md
├── 文档2.md
├── 子目录/
│   ├── 文档3.md
│   └── 文档4.md
└── assets/           # 图片等资源文件
    ├── image1.png
    └── image2.jpg
```

## API 接口

本功能使用思源笔记官方 API:

- `/api/notebook/lsNotebooks` - 获取笔记本列表
- `/api/export/exportNotebookMd` - 导出笔记本为 Markdown

## 配置选项

在插件设置中可以:

- 启用/禁用 Markdown 导出功能
- 设置是否显示顶部工具栏按钮

## 注意事项

1. **大笔记本导出时间**: 如果笔记本文档较多,导出可能需要一些时间,请耐心等待
2. **存储空间**: 确保有足够的磁盘空间存放导出的 ZIP 文件
3. **网络连接**: 如果思源笔记运行在远程服务器,需要稳定的网络连接
4. **权限要求**: 需要有笔记本的读取权限

## 常见问题

### Q: 导出失败怎么办?

A: 检查以下几点:
- 思源笔记服务是否正常运行
- 是否有笔记本的读取权限
- 查看浏览器控制台的错误日志

### Q: 导出的文件在哪里?

A: 导出的 ZIP 文件会自动下载到浏览器的默认下载目录

### Q: 可以导出单个文档吗?

A: 当前版本仅支持导出整个笔记本,暂不支持单个文档导出
