# skillLearning - 技能学习

编程技能闪卡学习功能，支持卡片管理、选择题测验、间隔重复复习、学习统计。

## 视图

- **列表视图** - 浏览、搜索、筛选、分页查看所有卡片
- **闪卡视图** - 随机选择题测验，支持键盘快捷键
- **复习视图** - 基于 SM-2 算法的间隔重复复习
- **统计视图** - 练习覆盖、正确率、语言/难度/分类分布

## 数据存储

卡片数据持久化在思源插件存储中，使用 `PluginStorage` 统一存储层。首次加载时自动导入 20 张内置预设卡片（C# / JavaScript / TypeScript / Vue 各 5 张）。
