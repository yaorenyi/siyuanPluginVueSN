# 本地磁盘浏览器

在思源笔记中浏览本地磁盘和文件夹。注册为右侧边栏 Dock 面板，显示本地磁盘驱动器列表、文件夹层级树、面包屑导航，支持收藏文件夹、复制路径、查看文件夹大小和修改时间等。

## 使用方法

1. 在设置中启用「本地磁盘浏览器」
2. 右侧边栏 Dock 面板中点击磁盘卡片展开
3. 双击文件夹进入子目录，双击文件在资源管理器中打开
4. 点击星标按钮收藏常用文件夹
5. 通过面包屑导航快速跳转任意层级
6. 点击刷新按钮获取最新数据

## 架构

- **Dock 面板** — `createVueDockApp` 注册为右侧边栏
- **缓存策略** — 磁盘列表和文件夹内容分别缓存，1 小时过期
- **并发控制** — PowerShell 命令通过防抖队列顺序执行
- **持久化** — 收藏夹路径通过 PluginStorage / TypedStorage 保存
