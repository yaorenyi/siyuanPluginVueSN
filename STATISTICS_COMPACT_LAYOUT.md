# 统计面板紧凑布局优化

## 优化目标
减少顶部统计卡片的占用空间，提高信息密度，让界面更加紧凑高效。

## 布局对比

### 优化前（占用空间大）
```
┌─────────────┬─────────────┐
│   12,345    │   456,789   │  ← 两个大卡片
│ 📓 笔记总数  │ ✍️ 总字数    │
└─────────────┴─────────────┘

┌────┬────┬────┬────┐
│ 🧩 │ 📎 │ 🏷️ │ 🔗 │  ← 四个小卡片
│123 │ 45 │ 67 │ 89 │
└────┴────┴────┴────┘
```
**占用高度**：约 120px

### 优化后（紧凑布局）
```
┌──────────────────────────────────────┐
│ 📓 12,345  │  ✍️ 456,789              │  ← 一个卡片，左右分栏
│   笔记总数  │    总字数                │
└──────────────────────────────────────┘

┌──────────────────────────────────────┐
│ 🧩 123 内容块 │ 📎 45 附件 │ 🏷️ 67 标签 │ 🔗 89 双链 │  ← 一行显示
└──────────────────────────────────────┘
```
**占用高度**：约 80px  
**节省空间**：约 33%

## 设计细节

### 1. 主要统计卡片
**布局：**
- 单个卡片，内部左右分栏
- 中间添加分隔线
- 图标放大到 32px
- 数值和标签垂直排列

**样式：**
- 紫色渐变背景（保持原有风格）
- 白色文字
- 悬停上浮效果

**结构：**
```
[图标 数值/标签] | [图标 数值/标签]
```

### 2. 次要统计卡片
**布局：**
- 单行横向排列
- 四个统计项均匀分布
- 图标 + 数值 + 标签水平排列

**样式：**
- 浅色背景（主题色）
- 淡边框
- 数值使用主题强调色
- 悬停边框高亮

**结构：**
```
[图标 数值 标签] [图标 数值 标签] [图标 数值 标签] [图标 数值 标签]
```

## 视觉层次

### 信息优先级
1. **最重要**：笔记总数、总字数（大卡片，渐变背景）
2. **次要**：内容块、附件、标签、双链（小卡片，简洁样式）
3. **辅助**：今日新增、今日修改、平均字数（独立卡片组）

### 颜色方案
- **主卡片**：紫色渐变 (#667eea → #764ba2)
- **次卡片**：主题背景色 + 边框
- **强调色**：主题蓝色（数值）

## 响应式设计

### 宽度适配
- **宽屏**：所有元素正常显示
- **窄屏**：次要统计自动换行或缩小间距

### 字体大小
- 主卡片数值：24px
- 主卡片标签：11px
- 次卡片数值：16px
- 次卡片标签：10px
