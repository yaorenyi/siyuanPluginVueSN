# 内容加密功能

## 功能概述

新增内容加密功能，支持对选中文本进行加密和解密操作，使用 AES-256-GCM 加密算法确保安全性。

## 功能特点

1. **安全加密算法**：使用 AES-256-GCM 加密算法
2. **右键菜单集成**：选中文本后右键即可加密/解密
3. **密码管理**：在通用设置中统一管理加密密码
4. **超级面板开关**：可在超级面板中启用/禁用功能
5. **持久化存储**：使用思源 API 持久化密码配置
6. **优化的解密UI**：
   - 直观的两步式界面（输入密码 → 显示结果）
   - 解密后可预览内容
   - 提供复制和替换两种操作选项
   - 不点击替换按钮则不会修改原文

## 使用方法

### 1. 设置密码

1. 打开右侧边栏的"通用设置"面板
2. 选择"加密设置"标签
3. 输入新密码并确认
4. 点击"保存"按钮

### 2. 加密文本

1. 在编辑器中选中要加密的文本
2. 右键打开菜单
3. 点击"🔒 加密文本"
4. 文本将被加密并以 `[encrypted]...[/encrypted]` 格式显示

### 3. 解密文本

1. 选中加密的文本（包含 `[encrypted]...[/encrypted]` 标记）
2. 右键打开菜单
3. 点击"🔓 解密文本"
4. 在弹出的对话框中输入密码
5. 点击"解密"按钮
6. 解密成功后，对话框会显示解密内容
7. 可以选择：
   - **复制内容**：将解密内容复制到剪贴板
   - **替换加密文本**：用解密内容替换原来的加密文本
   - **关闭**：不做任何操作，关闭对话框

### 4. 启用/禁用功能

1. 打开超级面板
2. 找到"内容加密"功能卡片
3. 点击开关按钮启用或禁用

### 核心功能

1. **加密模块** (`src/features/encryption/index.ts`)
   - 右键菜单注册
   - AES-256-GCM 加密/解密
   - 密码管理
   - 思源 API 持久化

2. **密码设置组件** (`EncryptionSettings.vue`)
   - 密码状态显示
   - 密码输入和确认
   - 密码保存

3. **配置集成**
   - 超级面板功能卡片
   - 通用设置标签页
   - 插件配置项

## 安全说明

- 使用 AES-256-GCM 加密算法，安全可靠
- 密码使用 PBKDF2 派生密钥，迭代 100,000 次
- 每次加密使用随机 IV（初始化向量）
- 加密数据使用 Base64 编码存储

## 注意事项

1. 请妥善保管密码，忘记密码将无法解密
2. 加密文本格式：`[encrypted]Base64EncodedData[/encrypted]`
3. 不要手动修改加密文本，否则可能导致解密失败
4. 建议定期备份重要的加密内容
