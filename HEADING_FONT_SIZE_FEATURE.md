# H1-H6标题字体大小设置功能

## 功能概述

为标题设置模块新增了H1-H6所有标题级别的字体大小自定义功能，允许用户根据个人喜好调整所有标题的显示大小。

## 新增功能

### 1. 通用字体大小范围
- **调节范围**: 14px - 40px（所有标题级别）
- **步长**: 1px
- **实时预览**: 滑块调节时显示当前像素值

### 2. 各标题级别默认值
- **H1**: 28px（文档主标题）
- **H2**: 24px（章节标题）
- **H3**: 20px（子章节标题）
- **H4**: 18px（小节标题）
- **H5**: 16px（段落标题）
- **H6**: 14px（最小标题）

## UI设计

### 视觉特性
- 🎨 **渐变图标**: 每个标题级别都有独特的渐变色彩
  - H1: 红色渐变 (#F39A94 → #E57373)
  - H2: 橙色渐变 (#F8D694 → #FFB74D)
  - H3: 绿色渐变 (#B1DCB9 → #81C784)
  - H4: 蓝色渐变 (#AAD2FC → #64B5F6)
  - H5: 紫色渐变 (#AC9DC0 → #9575CD)
  - H6: 灰色渐变 (#D7D7D7 → #9E9E9E)
- 📏 **滑块控制**: 直观的滑块界面，支持精确调节
- 🎯 **实时反馈**: 调节时立即显示数值变化
- 📱 **响应式**: 适配不同屏幕尺寸

### 交互体验
- 悬停效果增强视觉反馈
- 平滑的过渡动画
- 清晰的数值标识
- 颜色编码区分不同标题级别

## 使用方法

### 1. 打开设置
1. 点击右侧边栏的"通用设置"
2. 选择"标题配置"选项卡

### 2. 调节字体大小
1. 找到"标题字体大小"设置区域
2. 使用滑块调节各个级别的标题大小（14px-40px）
   - H1标题：20px-40px，默认28px
   - H2标题：14px-40px，默认24px
   - H3标题：14px-40px，默认20px
   - H4标题：14px-40px，默认18px
   - H5标题：14px-40px，默认16px
   - H6标题：14px-40px，默认14px
3. 观察预览区域的实时效果

### 3. 保存设置
- 设置会自动保存
- 重启插件或思源后设置依然生效

## 预览功能

### 实时预览
- 预览区域即时显示所有标题级别的字体大小变化
- 所有H1-H6标题都会根据滑块值动态调整
- 可以同时看到六个标题级别的层次关系

### 视觉对比
- 可以同时看到不同级别的标题效果
- 便于选择合适的字体大小搭配

## 最佳实践

### 建议设置
- **文档标题**: H1设置为32-36px，突出重要性
- **章节标题**: H2设置为24-28px，层次分明
- **阅读体验**: 考虑屏幕尺寸和阅读距离

### 设计原则
- 保持层级关系：H1 > H2
- 考虑响应式：移动端适当缩小
- 注重可读性：避免过大或过小

## 故障排除

### 字体大小未生效
1. 检查是否保存了设置
2. 重新加载插件或重启思源
3. 清除浏览器缓存

### 预览异常
1. 确认滑块值在有效范围内
2. 检查CSS样式是否被其他插件覆盖
3. 重置为默认设置重新配置
