# 2026-06-12

## diskBrowser 模块代码审查与重构

对 `src/features/diskBrowser/` 模块进行了全面的代码审查与重构，完成 7 项改造：

1. **Storage 双重实例化修复**（CLAUDE.md 违规）：将 `DiskBrowserStorage` 改为单例模式，通过 `createVueDockApp` 的 `extraProps` 传入 composable，消除 `index.ts` 和 `useDiskBrowser.ts` 各创建一个实例的问题。

2. **Vue 事件命名规范**：`FolderList.vue` 中 `item-dblclick`、`toggle-favorite`、`copy-path` 改为 camelCase（`itemDblclick`、`toggleFavorite`、`copyPath`），符合 CLAUDE.md「emit 事件必须 camelCase」规则。

3. **删除死代码**：移除 `utils/index.ts` 中的 `isNetworkSlow()` 函数及其模块级缓存变量。该函数检查 `navigator.connection.effectiveType`，但 diskBrowser 访问本地磁盘完全不涉及网络。`getCacheExpiryTime()` 简化为直接返回 60 分钟。

4. **favoriteSet 改为 computed**：从独立 `ref(new Set<string>())` + 手动 `updateFavoriteSet()` 同步改为 `computed(() => new Set(favoriteFolders.value))`，消除手动同步代码和不同步风险。

5. **Spin-wait → Promise 链式队列**：`retryExec()` 中 `while (isExecutingCommand) { await setTimeout(100) }` 忙等待改为 Promise 链式队列（`execQueue.then(...)`），消除轮询开销。

6. **SCSS 样式去重**：提取 `@mixin cache-tag-base`，`PanelHeader.vue` 的 `.cache-tag` 和 `FolderList.vue` 的 `.cache-tag-small` 共用基础样式，各自仅覆盖差异属性。

7. **类型断言 + 全局样式**：`index.ts` 使用 `{ diskBrowser: DiskBrowserI18n }` 替代 `as any`；`styles/index.scss` 添加 `@use "@/index.scss" as *;`。

构建验证通过，无 lint 错误。

## SkillsModal.vue 审查修复 + Codex 样式重构

对 `src/features/floatingBox/components/SkillsModal.vue` 和 `styles/index.scss` 进行了全面重构，包含 9 项修复：

1. **plugin 类型安全**：`props.plugin` 从 `any` 改为 `Plugin`（来自 `siyuan` 包）。
2. **storage 实例化修复（Critical）**：`storage` 从 `computed(() => new FloatingBoxStorage(...))` 改为 `ref<FloatingBoxStorage | null>(null)`，在 `onMounted` 中仅初始化一次，消除每次 `storage.value` 访问创建新实例的性能问题。
3. **原生弹窗替换**：7 处 `alert()/confirm()` 全部替换为 `showMessage(msg, duration, type)` from `siyuan`（项目标准通知 API）。
4. **category 预缓存**：使用 `categoryMap` computed（`Map<string, SkillCategory>`）实现 O(1) 查找；`filteredSkills` 返回 `SkillDisplay` 增强对象，预计算 `catName`/`catColor`/`catBgColor`，消除每卡片 3 次模板函数调用。
5. **v-for 消除模板重复**：3 块 content 复制区从复制粘贴改为 `skill.contents` 数组 + `v-for` 遍历，配套 `ContentSlot` 接口定义。
6. **搜索范围扩展**：`filteredSkills` 新增 `content2`/`content3` 字段搜索。
7. **键盘处理**：3 个 Modal overlay 添加 `@keydown.escape` 关闭；content 复制区 `role="button"` 添加 `@keydown.enter` + `@keydown.space.prevent`。
8. **loading 状态**：添加 `loading` ref，onMounted 期间显示加载提示。
9. **Codex (vp-*) 样式重构**：SCSS 类名从 `skills-modal-*`/`skill-*` 迁移为 `vp-*` 体系（vp-overlay/vp-modal/vp-card/vp-tag/vp-content-block 等）；代码内容使用 `$vp-mono` 等宽字体；表单标签使用代码风格（uppercase/opacity 0.45/letter-spacing）；焦点发光对齐 `var(--b3-theme-primary-lightest)`；移除空 `.trigger-icon {}` 规则；修复 `.category-item:hover` 缩进 bug。
