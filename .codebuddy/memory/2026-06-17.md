## Bug 修复：字体族设置不生效

**根因**：动态 CSS (`styles.ts` `applyCodeBlockEnhancedStyles()`) 只设置了 `.code-block .hljs` 的 `font-family`，但思源编辑器的 `.hljs > code` 子元素自带 `font-family`，阻断了继承。

**修复** (`src/features/generalSettings/utils/styles.ts`)：
- 编辑区：`.protyle-wysiwyg .code-block .hljs` → 新增 `.hljs code` 和 `.hljs span` 选择器
- 预览区：`.b3-typography pre` / `pre code` → 新增 `pre span` 选择器
- `font-family` 拆分到独立规则块中（避免和其他属性的 `!important` 混在一起）

## 颜色选择卡顿修复 + 透明度支持

**卡顿根因**：`applyCodeBlockEnhancedStyles` 每次设置变更时移除+重建完整 `<style>` 元素（~140 行 CSS），拖动颜色选择器时高频触发。

**修复** (`CodeBlockSettings.vue`)：
- `applyCodeBlockStyle` 和 `applyCodeBlockCollapse` 保持立即执行（仅操作 class/style toggle）
- `applyCodeBlockEnhancedStyles` 增加 100ms 防抖（`debouncedApplyEnhanced`）

**透明度** (3 文件)：
- `storage.ts`：`CodeBlockSettings` 新增 `backgroundColorOpacity: number` (0-1)，默认 1
- `styles.ts`：新增 `hexToRgba()` 辅助函数，`opacity >= 1` 直接返回 hex，否则转 `rgba(r, g, b, o)`
- `CodeBlockSettings.vue`：背景色区域新增 10%-100% 范围滑块，带 +/− 按钮和百分比显示
