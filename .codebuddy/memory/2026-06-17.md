### DocCountManager setBoxCount 批量查询优化
原 N 个笔记本 → N 次串行 SQL（`for...await fetch`），每轮网络往返 ~50ms。
优化为 1 次 `SELECT box, count(*) FROM blocks WHERE box IN (...) AND type = 'd' GROUP BY box`，然后构建 `countMap` 索引遍历 DOM。
10 笔记本场景 ~500ms → ~50ms（约 10x 提升）。
