# 2026-06-18

## GitPush index.vue 拆分重构

### 目标
将 2483 行的 index.vue 拆分为 3 个 composable + 7 个弹窗子组件，缩减至约 1200 行。

### 变更

**新增 composable（3 个）**：
- `composables/useTimeUtils.ts` — `relativeTime()`, `activityLevel()`, `sortProjects()` 纯工具函数
- `composables/useProjectFilters.ts` — `searchQuery`, `viewMode`, `showArchived`, `gitOpsPaused`, `selectedTags`, `smartViewProjects`, `filteredGroups`
- `composables/useIdeManagement.ts` — IDE_DETECTION + IDE_PRESETS + `scanIdes()`, `handleOpenIde()`, `launchIde()`, `isCmdAvailable()`, `resolvePath()`, `findSlnFile()`, `addCustomIde()`, `doRemoveCustomIde()`, `loadCustomIdes()`, `saveCustomIdes()` — ~230 行

**新增弹窗组件（7 个）**：
- `components/AddProjectDialog.vue` — 添加项目弹窗（props: i18n, categories; emit: add）
- `components/CategoryDialog.vue` — 分类管理弹窗（emit: add-category, delete-category）
- `components/SettingsDialog.vue` — 设置弹窗（props: concurrency; emit: save）
- `components/EditProjectDialog.vue` — 编辑项目弹窗（含远程仓库管理）（props: project, remoteList, allTags）
- `components/ScanImportDialog.vue` — 扫描导入弹窗（props: scanning, results, selection）
- `components/IdeManagementDialog.vue` — IDE 管理弹窗（props: customIdes, presetOptions）
- `components/RemoteConfigDialog.vue` — 远程仓库配置弹窗（emit: add-remote, remove-remote, edit-remote）

**跳过 ProjectCard 拆分**：30+ props + 25+ emits 的 prop drilling 反而降低可维护性，弹窗拆分 + composable 提取已充满足目标。

### 构建验证
- `npx vue-tsc --noEmit` 通过
- `npx vite build` 通过
- 零新增 lint 错误
- 通信模式：弹窗通过 props/emit 与 index.vue 解耦
- 所有 composable 内部使用 `getNodeProcessModules()` / `getNodeModules()` 不再通过 index.vue 传递
